# 上经三十卦详解

上经三十卦始于乾坤，终于坎离，以天地之道为主。

---

## 第1卦 乾 ䷀ 乾为天

**卦辞**：元，亨，利，贞。

**彖传**：大哉乾元，万物资始，乃统天。云行雨施，品物流形。大明终始，六位时成，时乘六龙以御天。乾道变化，各正性命，保合太和，乃利贞。首出庶物，万国咸宁。

**大象**：天行健，君子以自强不息。

| 爻 | 爻辞 | 小象 |
|----|------|------|
| 初九 | 潜龙勿用 | 阳在下也 |
| 九二 | 见龙在田，利见大人 | 德施普也 |
| 九三 | 君子终日乾乾，夕惕若厉，无咎 | 反复道也 |
| 九四 | 或跃在渊，无咎 | 进无咎也 |
| 九五 | 飞龙在天，利见大人 | 大人造也 |
| 上九 | 亢龙有悔 | 盈不可久也 |
| 用九 | 见群龙无首，吉 | 天德不可为首也 |

---

## 第2卦 坤 ䷁ 坤为地

**卦辞**：元，亨，利牝马之贞。君子有攸往，先迷后得主，利。西南得朋，东北丧朋。安贞吉。

**彖传**：至哉坤元，万物资生，乃顺承天。坤厚载物，德合无疆。含弘光大，品物咸亨。

**大象**：地势坤，君子以厚德载物。

| 爻 | 爻辞 | 小象 |
|----|------|------|
| 初六 | 履霜，坚冰至 | 阴始凝也 |
| 六二 | 直方大，不习无不利 | 地道光也 |
| 六三 | 含章可贞，或从王事，无成有终 | 知光大也 |
| 六四 | 括囊，无咎无誉 | 慎不害也 |
| 六五 | 黄裳，元吉 | 文在中也 |
| 上六 | 龙战于野，其血玄黄 | 其道穷也 |
| 用六 | 利永贞 | 以大终也 |

---

## 第3卦 屯 ䷂ 水雷屯

**卦辞**：元，亨，利，贞。勿用有攸往。利建侯。

**彖传**：屯，刚柔始交而难生。动乎险中，大亨贞。

**大象**：云雷屯，君子以经纶。

| 爻 | 爻辞 |
|----|------|
| 初九 | 磐桓，利居贞，利建侯 |
| 六二 | 屯如邅如，乘马班如。匪寇婚媾，女子贞不字，十年乃字 |
| 六三 | 即鹿无虞，惟入于林中，君子几不如舍，往吝 |
| 六四 | 乘马班如，求婚媾，往吉，无不利 |
| 九五 | 屯其膏，小贞吉，大贞凶 |
| 上六 | 乘马班如，泣血涟如 |

---

## 第4卦 蒙 ䷃ 山水蒙

**卦辞**：亨。匪我求童蒙，童蒙求我。初筮告，再三渎，渎则不告。利贞。

**彖传**：蒙，山下有险，险而止，蒙。

**大象**：山下出泉，蒙。君子以果行育德。

| 爻 | 爻辞 |
|----|------|
| 初六 | 发蒙，利用刑人，用说桎梏，以往吝 |
| 九二 | 包蒙，吉。纳妇，吉。子克家 |
| 六三 | 勿用取女，见金夫，不有躬，无攸利 |
| 六四 | 困蒙，吝 |
| 六五 | 童蒙，吉 |
| 上九 | 击蒙，不利为寇，利御寇 |

---

## 第5卦 需 ䷄ 水天需

**卦辞**：有孚，光亨，贞吉。利涉大川。

**彖传**：需，须也，险在前也。刚健而不陷，其义不困穷矣。

**大象**：云上于天，需。君子以饮食宴乐。

| 爻 | 爻辞 |
|----|------|
| 初九 | 需于郊，利用恒，无咎 |
| 九二 | 需于沙，小有言，终吉 |
| 九三 | 需于泥，致寇至 |
| 六四 | 需于血，出自穴 |
| 九五 | 需于酒食，贞吉 |
| 上六 | 入于穴，有不速之客三人来，敬之终吉 |

---

## 第6卦 讼 ䷅ 天水讼

**卦辞**：有孚窒惕，中吉，终凶。利见大人，不利涉大川。

**彖传**：讼，上刚下险，险而健，讼。

**大象**：天与水违行，讼。君子以作事谋始。

| 爻 | 爻辞 |
|----|------|
| 初六 | 不永所事，小有言，终吉 |
| 九二 | 不克讼，归而逋，其邑人三百户，无眚 |
| 六三 | 食旧德，贞厉，终吉。或从王事，无成 |
| 九四 | 不克讼，复即命，渝安贞，吉 |
| 九五 | 讼，元吉 |
| 上九 | 或锡之鞶带，终朝三褫之 |

---

## 第7卦 师 ䷆ 地水师

**卦辞**：贞，丈人吉，无咎。

**彖传**：师，众也。贞，正也。能以众正，可以王矣。

**大象**：地中有水，师。君子以容民畜众。

| 爻 | 爻辞 |
|----|------|
| 初六 | 师出以律，否臧凶 |
| 九二 | 在师中，吉无咎，王三锡命 |
| 六三 | 师或舆尸，凶 |
| 六四 | 师左次，无咎 |
| 六五 | 田有禽，利执言，无咎。长子帅师，弟子舆尸，贞凶 |
| 上六 | 大君有命，开国承家，小人勿用 |

---

## 第8卦 比 ䷇ 水地比

**卦辞**：吉。原筮，元永贞，无咎。不宁方来，后夫凶。

**彖传**：比，吉也。比，辅也，下顺从也。

**大象**：地上有水，比。先王以建万国，亲诸侯。

| 爻 | 爻辞 |
|----|------|
| 初六 | 有孚比之，无咎。有孚盈缶，终来有它，吉 |
| 六二 | 比之自内，贞吉 |
| 六三 | 比之匪人 |
| 六四 | 外比之，贞吉 |
| 九五 | 显比，王用三驱，失前禽，邑人不诫，吉 |
| 上六 | 比之无首，凶 |

---

## 第9卦 小畜 ䷈ 风天小畜

**卦辞**：亨。密云不雨，自我西郊。

**彖传**：小畜，柔得位而上下应之，曰小畜。

**大象**：风行天上，小畜。君子以懿文德。

| 爻 | 爻辞 |
|----|------|
| 初九 | 复自道，何其咎，吉 |
| 九二 | 牵复，吉 |
| 九三 | 舆说辐，夫妻反目 |
| 六四 | 有孚，血去惕出，无咎 |
| 九五 | 有孚挛如，富以其邻 |
| 上九 | 既雨既处，尚德载，妇贞厉。月几望，君子征凶 |

---

## 第10卦 履 ䷉ 天泽履

**卦辞**：履虎尾，不咥人，亨。

**彖传**：履，柔履刚也。说而应乎乾。

**大象**：上天下泽，履。君子以辩上下，定民志。

| 爻 | 爻辞 |
|----|------|
| 初九 | 素履，往无咎 |
| 九二 | 履道坦坦，幽人贞吉 |
| 六三 | 眇能视，跛能履，履虎尾咥人，凶。武人为于大君 |
| 九四 | 履虎尾，愬愬终吉 |
| 九五 | 夬履，贞厉 |
| 上九 | 视履考祥，其旋元吉 |

---

## 第11卦 泰 ䷊ 地天泰

**卦辞**：小往大来，吉，亨。

**彖传**：泰，小往大来，吉亨。则是天地交而万物通也，上下交而其志同也。

**大象**：天地交，泰。后以财成天地之道，辅相天地之宜，以左右民。

| 爻 | 爻辞 |
|----|------|
| 初九 | 拔茅茹，以其汇，征吉 |
| 九二 | 包荒，用冯河，不遐遗，朋亡，得尚于中行 |
| 九三 | 无平不陂，无往不复，艰贞无咎。勿恤其孚，于食有福 |
| 六四 | 翩翩，不富以其邻，不戒以孚 |
| 六五 | 帝乙归妹，以祉元吉 |
| 上六 | 城复于隍，勿用师，自邑告命，贞吝 |

---

## 第12卦 否 ䷋ 天地否

**卦辞**：否之匪人，不利君子贞。大往小来。

**彖传**：否之匪人，不利君子贞。则是天地不交而万物不通也。

**大象**：天地不交，否。君子以俭德辟难，不可荣以禄。

| 爻 | 爻辞 |
|----|------|
| 初六 | 拔茅茹，以其汇，贞吉亨 |
| 六二 | 包承，小人吉，大人否亨 |
| 六三 | 包羞 |
| 九四 | 有命无咎，畴离祉 |
| 九五 | 休否，大人吉。其亡其亡，系于苞桑 |
| 上九 | 倾否，先否后喜 |

---

## 第13卦 同人 ䷌ 天火同人

**卦辞**：同人于野，亨。利涉大川，利君子贞。

**彖传**：同人，柔得位得中而应乎乾，曰同人。

**大象**：天与火，同人。君子以类族辨物。

| 爻 | 爻辞 |
|----|------|
| 初九 | 同人于门，无咎 |
| 六二 | 同人于宗，吝 |
| 九三 | 伏戎于莽，升其高陵，三岁不兴 |
| 九四 | 乘其墉，弗克攻，吉 |
| 九五 | 同人先号咷而后笑，大师克相遇 |
| 上九 | 同人于郊，无悔 |

**卦辞**：元亨。

**彖传**：大有，柔得尊位大中而上下应之，曰大有。

**大象**：火在天上，大有。君子以遏恶扬善，顺天休命。

| 九二 | 大车以载，有攸往，无咎 |
| 九三 | 公用亨于天子，小人弗克 |
| 九四 | 匪其彭，无咎 |
| 六五 | 厥孚交如，威如，吉 |
| 上九 | 自天佑之，吉无不利 |

**卦辞**：亨，君子有终。

**彖传**：谦，亨。天道下济而光明，地道卑而上行。

**大象**：地中有山，谦。君子以裒多益寡，称物平施。

| 爻 | 爻辞 |
|----|------|
| 初六 | 谦谦君子，用涉大川，吉 |
| 六二 | 鸣谦，贞吉 |
| 九三 | 劳谦君子，有终吉 |
| 六四 | 无不利，撝谦 |
| 六五 | 不富以其邻，利用侵伐，无不利 |
| 上六 | 鸣谦，利用行师，征邑国 |

**彖传**：豫，刚应而志行，顺以动，豫。

| 六五 | 贞疾，恒不死 |
| 上六 | 冥豫，成有渝，无咎 |
