# 下经三十四卦详解

下经始于咸恒，终于既济未济，以人伦人事为主。

---

## 第31卦 咸 ䷞ 泽山咸

**卦辞**：亨，利贞。取女吉。

**彖传**：咸，感也。柔上而刚下，二气感应以相与。止而说，男下女，是以亨，利贞，取女吉也。天地感而万物化生，圣人感人心而天下和平。观其所感，而天地万物之情可见矣。

**大象**：山上有泽，咸。君子以虚受人。

| 爻 | 爻辞 |
|----|------|
| 初六 | 咸其拇 |
| 六二 | 咸其腓，凶，居吉 |
| 九三 | 咸其股，执其随，往吝 |
| 九四 | 贞吉悔亡，憧憧往来，朋从尔思 |
| 九五 | 咸其脢，无悔 |
| 上六 | 咸其辅颊舌 |

## 第32卦 恒 ䷟ 雷风恒

**卦辞**：亨，无咎，利贞。利有攸往。

**彖传**：恒，久也。刚上而柔下，雷风相与，巽而动，刚柔皆应，恒。

**大象**：雷风，恒。君子以立不易方。

| 九二 | 悔亡 |
| 九三 | 不恒其德，或承之羞，贞吝 |
| 九四 | 田无禽 |
| 六五 | 恒其德，贞。妇人吉，夫子凶 |
| 上六 | 振恒，凶 |

## 第33卦 遁 ䷠ 天山遁

**卦辞**：亨，小利贞。

**彖传**：遁亨，遁而亨也。刚当位而应，与时行也。小利贞，浸而长也。遁之时义大矣哉！

**大象**：天下有山，遁。君子以远小人，不恶而严。

| 九四 | 好遁，君子吉，小人否 |
| 九五 | 嘉遁，贞吉 |
| 上九 | 肥遁，无不利 |

## 第34卦 大壮 ䷡ 雷天大壮

**卦辞**：利贞。

**彖传**：大壮，大者壮也。刚以动，故壮。大壮利贞，大者正也。正大而天地之情可见矣。

**大象**：雷在天上，大壮。君子以非礼弗履。

| 九四 | 贞吉悔亡。藩决不羸，壮于大舆之輹 |
| 六五 | 丧羊于易，无悔 |
| 上六 | 羝羊触藩，不能退，不能遂，无攸利，艰则吉 |

## 第36卦 明夷 ䷣ 地火明夷

**卦辞**：利艰贞。

**彖传**：明入地中，明夷。内文明而外柔顺，以蒙大难，文王以之。利艰贞，晦其明也。内难而能正其志，箕子以之。

**大象**：明入地中，明夷。君子以莅众，用晦而明。

| 爻 | 爻辞 |
|----|------|

| 六二 | 明夷，夷于左股，用拯马壮，吉 |
| 九三 | 明夷于南狩，得其大首，不可疾贞 |
| 六四 | 入于左腹，获明夷之心，于出门庭 |
| 六五 | 箕子之明夷，利贞 |
| 上六 | 不明晦，初登于天，后入于地 |

## 第37卦 家人 ䷤ 风火家人

**卦辞**：利女贞。

**彖传**：家人，女正位乎内，男正位乎外。男女正，天地之大义也。家人有严君焉，父母之谓也。父父子子兄兄弟弟夫夫妇妇而家道正，正家而天下定矣。

**大象**：风自火出，家人。君子以言有物而行有恒。

|----|------|

| 六四 | 富家，大吉 |
| 九五 | 王假有家，勿恤吉 |
| 上九 | 有孚威如，终吉 |

## 第38卦 睽 ䷥ 火泽睽

**卦辞**：小事吉。

**彖传**：睽，火动而上，泽动而下。二女同居，其志不同行。说而丽乎明，柔进而上行，得中而应乎刚，是以小事吉。天地睽而其事同也，男女睽而其志通也，万物睽而其事类也。睽之时用大矣哉！

**大象**：上火下泽，睽。君子以同而异。

| 爻 | 爻辞 |
|----|------|
| 初九 | 悔亡，丧马勿逐自复，见恶人无咎 |
| 九二 | 遇主于巷，无咎 |
| 六三 | 见舆曳，其牛掣，其人天且劓，无初有终 |
| 九四 | 睽孤，遇元夫，交孚，厉无咎 |
| 六五 | 悔亡，厥宗噬肤，往何咎 |
| 上九 | 睽孤，见豕负涂，载鬼一车，先张之弧，后说之弧，匪寇婚媾，往遇雨则吉 |

## 第39卦 蹇 ䷦ 水山蹇

**卦辞**：利西南，不利东北。利见大人，贞吉。

**彖传**：蹇，难也，险在前也。见险而能止，知矣哉！蹇利西南，往得中也。不利东北，其道穷也。

**大象**：山上有水，蹇。君子以反身修德。

| 爻 | 爻辞 |

| 六四 | 往蹇来连 |
| 九五 | 大蹇朋来 |
| 上六 | 往蹇来硕，吉，利见大人 |

## 第40卦 解 ䷧ 雷水解

**卦辞**：利西南。无所往，其来复吉。有攸往，夙吉。

**彖传**：解，险以动，动而免乎险，解。解利西南，往得众也。天地解而雷雨作，雷雨作而百果草木皆甲坼。解之时大矣哉！

**大象**：雷雨作，解。君子以赦过宥罪。

| 爻 | 爻辞 |
|----|------|
| 初六 | 无咎 |
| 九二 | 田获三狐，得黄矢，贞吉 |
| 六三 | 负且乘，致寇至，贞吝 |
| 九四 | 解而拇，朋至斯孚 |
| 六五 | 君子维有解，吉，有孚于小人 |
| 上六 | 公用射隼于高墉之上，获之无不利 |

## 第41卦 损 ䷨ 山泽损

**卦辞**：有孚，元吉，无咎，可贞。利有攸往。曷之用？二簋可用享。

**彖传**：损，损下益上，其道上行。损而有孚，元吉，无咎。损刚益柔有时，损益盈虚，与时偕行。

**大象**：山下有泽，损。君子以惩忿窒欲。

| 爻 | 爻辞 |
|----|------|
| 初九 | 已事遄往，无咎，酌损之 |
| 九二 | 利贞，征凶，弗损益之 |
| 六三 | 三人行则损一人，一人行则得其友 |
| 六四 | 损其疾，使遄有喜，无咎 |
| 六五 | 或益之十朋之龟，弗克违，元吉 |
| 上九 | 弗损益之，无咎，贞吉。利有攸往，得臣无家 |

## 第42卦 益 ䷩ 风雷益

**卦辞**：利有攸往，利涉大川。

**彖传**：益，损上益下，民说无疆。自上下下，其道大光。利有攸往，中正有庆。利涉大川，木道乃行。益动而巽，日进无疆。天施地生，其益无方。凡益之道，与时偕行。

**大象**：风雷，益。君子以见善则迁，有过则改。

| 爻 | 爻辞 |
|----|------|
| 初九 | 利用为大作，元吉，无咎 |
| 六二 | 或益之十朋之龟，弗克违，永贞吉。王用享于帝，吉 |
| 六三 | 益之用凶事，无咎。有孚中行，告公用圭 |
| 六四 | 中行，告公从，利用为依迁国 |
| 九五 | 有孚惠心，勿问元吉。有孚惠我德 |
| 上九 | 莫益之，或击之，立心勿恒，凶 |
