# 八卦基础

> 易有太极，是生两仪，两仪生四象，四象生八卦。——《系辞传》

## 太极 → 两仪 → 四象 → 八卦

```
              ┌── 乾 ☰ (太阳)
         ┌── 阳阳 ──┤
         │          └── 兑 ☱ (少阴)
  太极 ──┤
         │          ┌── 离 ☲ (少阳)
         └── 阴阳 ──┤
                    └── 震 ☳ (太阴)
```

实际上四象再加一爻生八卦：

| 四象 | +阳爻 | +阴爻 |
|------|-------|-------|
| 太阳 ⚌ | 乾 ☰ | 兑 ☱ |
| 少阴 ⚍ | 离 ☲ | 震 ☳ |
| 少阳 ⚎ | 巽 ☴ | 坎 ☵ |
| 太阴 ⚏ | 艮 ☶ | 坤 ☷ |

## 先天八卦（伏羲八卦）

```
            乾 ☰
       兑 ☱    巽 ☴
    离 ☲          坎 ☵
       震 ☳    艮 ☶
            坤 ☷
```

**卦序**：乾1 兑2 离3 震4 巽5 坎6 艮7 坤8

**口诀**：天地定位，山泽通气，雷风相薄，水火不相射。

## 后天八卦（文王八卦）

```
            离 ☲
       巽 ☴    坤 ☷
    震 ☳          兑 ☱
       艮 ☶    乾 ☰
            坎 ☵
```

**口诀**：帝出乎震，齐乎巽，相见乎离，致役乎坤，说言乎兑，战乎乾，劳乎坎，成言乎艮。

## 八卦与五行生克

| 相生 | 含义 |
|------|------|
| 乾兑(金) → 坎(水) | 金生水 |
| 坎(水) → 震巽(木) | 水生木 |
| 震巽(木) → 离(火) | 木生火 |
| 离(火) → 坤艮(土) | 火生土 |
| 坤艮(土) → 乾兑(金) | 土生金 |

| 相克 | 含义 |
|------|------|
| 乾兑(金) → 震巽(木) | 金克木 |
| 震巽(木) → 坤艮(土) | 木克土 |
| 坤艮(土) → 坎(水) | 土克水 |
| 坎(水) → 离(火) | 水克火 |
| 离(火) → 乾兑(金) | 火克金 |
