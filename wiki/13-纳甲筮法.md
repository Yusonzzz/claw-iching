# 纳甲筮法

> 纳甲筮法由汉代京房创立，是火珠林、六爻预测的根基。将天干地支、五行六亲、世应飞伏系统化装入卦中。

---

## 一、装卦总纲

### 八宫卦序
每宫八卦，从本宫卦开始，逐爻变化：

```
本宫(六世) → 一世 → 二世 → 三世 → 四世 → 五世 → 游魂(四世) → 归魂(三世)
```

### 纳甲口诀

- **乾**：子寅辰午申戌（内卦纳甲，外卦纳壬）
- **坤**：未巳卯丑亥酉（内卦纳乙，外卦纳癸）
- **震**：子寅辰午申戌（纳庚）
- **巽**：丑亥酉未巳卯（纳辛）
- **坎**：寅辰午申戌子（纳戊）
- **离**：卯丑亥酉未巳（纳己）
- **艮**：辰午申戌子寅（纳丙）
- **兑**：巳卯丑亥酉未（纳丁）

## 三、装六亲

以卦所属五行（八宫五行）为"我"，各爻干支五行为"他"：

| 关系 | 六亲 |
|------|------|
| 同我 | 兄弟 |
| 我生 | 子孙 |
| 生我 | 父母 |
| 我克 | 妻财 |
| 克我 | 官鬼 |

**例**：乾宫卦属金，爻为甲子（子=水），金生水→"我生"→子孙

## 四、定世应

世爻代表问卦人自己，应爻代表所问之事。

世应相隔两位（世爻确定后，应爻在隔两位的爻上）。

| 卦型 | 世爻位置 |
|------|---------|
| 八纯卦（六世） | 上爻 |
| 一世卦 | 初爻 |
| 二世卦 | 二爻 |
| 三世卦 | 三爻 |
| 四世卦 | 四爻 |
| 五世卦 | 五爻 |
| 游魂卦 | 四爻 |
| 归魂卦 | 三爻 |

## 五、装六兽

按起卦日干定六兽排布：

| 六兽 | 甲乙日 | 丙丁日 | 戊日 | 己日 | 庚辛日 | 壬癸日 |
|------|--------|--------|------|------|--------|--------|
| 初爻 | 青龙 | 朱雀 | 勾陈 | 螣蛇 | 白虎 | 玄武 |

六兽从初爻排至上爻，按五行相生顺序：
青龙→朱雀→勾陈→螣蛇→白虎→玄武

## 六、断卦要点

### 用神取法
- 问父母长辈 → 取父母爻
- 问兄弟朋友 → 取兄弟爻
- 问妻财财运 → 取妻财爻
- 问官讼工作 → 取官鬼爻
- 问子女晚辈 → 取子孙爻

### 用神旺衰判断
- **旺**：得日月生扶、临日月、动爻生
- **衰**：受日月克、旬空月破、入墓、受动爻克
- **有用**：旺相或用神化进神（化回头生）
- **无用**：休囚被克或用神化退神

### 大象吉凶
- 用神旺相，生世爻 → 大吉
- 用神旺相，克世爻（且世旺） → 先难后易
- 用神休囚，克世爻 → 凶
- 忌神动克用神 → 所求难成
