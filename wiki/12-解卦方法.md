# 解卦方法

## 解卦总纲

> 易有圣人之道四焉：以言者尚其辞，以动者尚其变，以制器者尚其象，以卜筮者尚其占。——《系辞传》

---

## 一、解卦步骤

### 先看卦象
1. **认卦名卦象**：哪两卦相重？上卦什么？下卦什么？
2. **看卦德**：上卦和下卦各是什么性情？（如：艮止坎险→水山蹇，止于险前）
3. **看大象**：整体卦形像什么？（如：颐䷚像张开的嘴，故为"养"）

### 再看卦辞
4. **卦辞**：文王对全卦的总体判断，含"亨利贞吉凶悔吝"等断语
5. **彖传**：孔子对卦辞的哲学发挥，看"时义"与"卦主"

### 三看爻位
6. **动爻**：哪一爻在变？这一爻的爻辞是什么？
7. **爻位关系**（见下方爻位篇）

### 综合判断
8. 将卦象、卦辞、爻辞、体用四者合一，结合所问之事给出判断

## 二、爻位精要

### 六爻分位
```
上爻 — 终、极、退、宗庙
五爻 — 君位、成功、领导（最尊贵）
四爻 — 大臣、近君、多惧
三爻 — 公卿、多凶、中位
二爻 — 大夫、多誉（下卦中位）
初爻 — 始、元、庶民
```

### 当位与不当位
- 初、三、五为**阳位**，阳爻居之曰"当位"（正）
- 二、四、上为**阴位**，阴爻居之曰"当位"（正）
- 反之则"不当位"，多有不吉

**口诀**：初三五，阳；二四上，阴。

### 中正
- **二爻**为下卦中位，**五爻**为上卦中位
- 中位比当位更重要！"中胜于正"
- 二爻阴爻（六二）则"既中且正"，最佳
- 五爻阳爻（九五）则"既中且正"，乾卦九五"飞龙在天"即此

### 承乘比应
| 关系 | 说明 |
|------|------|
| **承** | 下爻对上爻的承接，阴承阳为吉 |
| **乘** | 上爻对下爻的凌驾，阴乘阳为凶 |
| **比** | 相邻两爻的关系，阴阳相求为吉 |
| **应** | 初应四、二应五、三应上，阴阳相应为吉 |

### 爻变之例
- **一爻变**：以本卦变爻爻辞为主
- **二爻变**：以本卦二变爻之上爻爻辞为主
- **三爻变**：以本卦卦辞为主，兼看变卦
- **四爻变**：以变卦二不变爻之下爻爻辞为主
- **五爻变**：以变卦不变爻爻辞为主
- **六爻全变**：乾坤以用九用六为主，余卦以变卦卦辞为主
- **六爻不变**：以本卦卦辞为主

## 三、体用生克法（梅花易数核心）

### 体用之分
- **体卦**：代表问卦人自己（不动之卦）
- **用卦**：代表所问之事、对方、环境（动爻所在之卦）

### 生克关系
| 关系 | 含义 | 吉凶 |
|------|------|------|
| **用生体** | 外部环境有利于你，有贵人相助 | 大吉 |
| **体生用** | 你需要付出努力才能成事 | 中平 |
| **比和** | 内外气场一致，事情顺遂 | 吉 |
| **体克用** | 你能掌控局面，但需要用力 | 中吉 |
| **用克体** | 外部压力大，事情难办 | 凶 |

## 四、互卦与变卦

### 互卦
从本卦中间四爻新组一卦：
- 本卦二、三、四爻 → 互卦下卦
- 本卦三、四、五爻 → 互卦上卦

互卦代表事情**发展过程中的内在变化**。

### 变卦（之卦）
动爻变化后形成的卦：
- 阳爻动 → 变阴爻
- 阴爻动 → 变阳爻

变卦代表事情的**最终走向和结果**。

---

## 五、卦象直读法（简捷入门）

不用背爻辞，直接看卦象：

1. **上卦看外部环境**，下卦看自身状态
2. **看上下卦五行生克**（如：上离火克下兑金→火泽睽，火克金，外部压制）
3. **看大象之形**（如：山雷颐䷚→口中含物→关于"养"的问题）
4. **结合卦名直断**（如：困䷮→陷入困境；泰䷊→通泰顺利）

## 六、动爻断事快查

| 动爻 | 含义 |
|------|------|
| 初爻动 | 事情刚开始变化，苗头初现 |
| 二爻动 | 内部开始变动，根基动摇 |
| 三爻动 | 事情进入关键转折点，吉凶参半 |
| 四爻动 | 外部环境开始影响，压力渐增 |
| 五爻动 | 高层/权威层面发生变化，成败在此一举 |
| 上爻动 | 事情接近尾声，或已过头，物极必反 |

---

## 七、占断注意事项

1. **心诚则灵**：杂念多时不占，酒醉不占，戏谑不占
2. **一事一占**：同一问题短期内不要反复占
3. **凶卦不惧**：卦象是警示不是定命，趋吉避凶才是智慧
4. **结合现实**：卦辞需结合具体问题解读，不可照搬字面
5. **初筮告，再三渎**：第一次占的结果最准确
