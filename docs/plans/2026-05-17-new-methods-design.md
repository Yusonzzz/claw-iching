# 新增起卦方式 + UI 优化 — 设计文档

## 设计上下文（Impeccable）

- **Target audience**: 关注传统文化、玄学、自我探索的中文用户
- **Use cases**: 快速获得易经卦象指引，体验不同起卦仪式感
- **Brand personality/tone**: 神秘优雅 · 东方韵味 · 现代简约 · 不廉价
- **Differentiation**: 起卦方式多元且每个都有独特的视觉交互形式，不是干巴巴的选择按钮

## 新增内容

### 1. 方位起卦

**原理（梅花易数·方位法）：**
八个方向映射八个卦：

| 方向 | 卦 | 数字 | 五行 |
|------|-----|------|------|
| 北 | 坎 | 1 | 水 |
| 东北 | 艮 | 7 | 土 |
| 东 | 震 | 4 | 木 |
| 东南 | 巽 | 5 | 木 |
| 南 | 离 | 3 | 火 |
| 西南 | 坤 | 8 | 土 |
| 西 | 兑 | 2 | 金 |
| 西北 | 乾 | 6 | 金 |

**算法：**
- 用户选择方向 → 上卦 = 方向的卦
- 下卦 = 当前时间（月+日）mod 8
- 动爻 = (方向数字 + 月 + 日) mod 6

**交互：**
- 底部弹出一个 iOS 风格的方向选择器
- 8个方向呈圆形排列或两排网格
- 每个方向有图标（箭头）和卦名标注

### 2. 物象起卦

**原理（梅花易数·观物法）：**
常见物体关联八卦：

| 类别 | 物体 | 卦 |
|------|------|-----|
| 天象 | 太阳、星星、雷电 | 乾/离/震 |
| 地象 | 山、水、风 | 艮/坎/巽 |
| 动物 | 龙马/鹿牛/鸡雉/猪鱼 | 乾/坤/离/坎 |
| 植物 | 松竹/花草 | 震/巽 |
| 器物 | 金玉/陶器/兵器/文书 | 乾/坤/离/巽 |

**算法：**
- 用户选择物体类别 → 选择具体物体 → 上卦 = 物体卦
- 下卦 = 选择序号 mod 8
- 动爻 = 总数 mod 6

**交互：**
- 先选类别（天/地/动物/植物/器物），再选具体物体
- 分两步的点选交互，有过渡动画

### 3. UI 优化

#### 起卦方式选择区域改造

**现状问题：** 3个方式在网格里挤在一起，新增2个后更挤

**改造方案：**
- 改成水平滑动滚轮（类似 iOS 的 PickerWheel 风格）
- 每项是一个卡片式按钮，带图标+名称+一句话说明
- 选中的方式居中放大，有高亮动画

#### 起卦方式参数区域优化

- 每个方式选中后，下方展开对应的输入区域（目前只有数字起卦有展开）
- 方位起卦 → 展开方向选择器
- 物象起卦 → 展开两级选择器

### 4. 内容深化

- 方位起卦的解读中，加入方向本身的文化含义（如：北=坎=险阻，南=离=光明）
- 物象起卦的解读中，加入物体象征意义的解释
